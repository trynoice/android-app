Bagi banyak orang, sedikit kebisingan latar belakang berguna untuk menenangkan diri dan fokus. Kebisingan latar belakang dapat membantu meminimalkan efek stres. Dalam beberapa kasus, ini juga meningkatkan produktivitas. Ini juga membantu orang dalam menutupi tinnitus mereka.

Noice adalah aplikasi yang memungkinkan Anda membuat suasana yang disesuaikan menggunakan klip dari sumber kebisingan lingkungan. Ciptakan lingkungan yang sempurna dengan menggabungkan suara yang berbeda untuk menghilangkan gangguan sehingga Anda dapat fokus pada pekerjaan Anda atau menghasilkan lingkungan yang damai untuk relaksasi.

<b>Fitur</b> 
- 35 suara yang direkam 
- Google Cast (Chromecast) diaktifkan (hanya di Play Store) 
- Pengatur waktu tidur otomatis 
- Jam Alarm 
- Hasilkan campuran acak 
- Buat campuran yang disesuaikan 
- Simpan campuran favorit Anda 
- Mainkan bersama pemutar musik lainnya 
- Kontrol volume individu untuk setiap sumber kebisingan 
- Pemutaran offline 
- Varian tema aplikasi terang dan gelap 

<b>Perpustakaan Noice</b> 
- Set kehidupan (burung, jangkrik, serigala, detak jantung manusia, dll.) 
- Set hutan (api unggun, malam, angin, pohon palem, dll.) 
- Set monsun (hujan, guntur) 
- Set Teleportasi (kedai kopi, perpustakaan, kantor, tepi laut, tepi sungai, dll.) 
- Set kendaraan (kereta bergerak, dalam penerbangan, kapal berderit, mobil listrik, dll.) 
- Mentahan noice (putih, merah muda, kecoklatan) 

<b>Izin</b> 
- <b>mencegah perangkat dari tidur</b>: untuk menjaga CPU perangkat tetap terjaga selama pemutaran 
- <b>jalankan layanan latar depan</b>: untuk menjalankan proses pemutar (berbeda dari UI) 
- <b>jalankan saat startup</b>: untuk memastikan timer bangun tetap ada selama reboot 
- <b>akses jaringan penuh</b>: untuk mengaktifkan Chromecast dan kontribusi keuangan 
- <b>melihat koneksi jaringan</b>: ditambahkan oleh ketergantungan, saat ini tidak digunakan 
- <b>Layanan penagihan Google Play</b>: untuk mengaktifkan kontribusi keuangan guna mendukung pengembangan 

Noice adalah perangkat lunak sumber terbuka dan gratis. https://github.com/ashutoshgngwr/noice
